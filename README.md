# Article Management System

A modern article management application built with React and Context API. The project simulates a lightweight content management system (CMS) where administrators can register, log in, create articles, and manage published content. All data is persisted using Local Storage, allowing the application to function without a backend service.

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/matinroghani/article-management.git
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
